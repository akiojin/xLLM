# CLI契約: safetensors量子化対応

**機能ID**: `SPEC-afa11b7d`
**作成日**: 2026-01-27

## 目的

CLIで量子化の指定および結果を確認できることを保証する。

## 期待される振る舞い

### 量子化指定

- safetensorsモデルに対して量子化方式を指定できること
- 未対応の量子化方式指定は明確なエラーを返すこと

### 表示

- `xllm show <MODEL>` で量子化情報が表示されること（適用時）
- `xllm list` で量子化が判別可能であること（列または詳細情報）

### エラー

- 未対応方式: "unsupported quantization" 相当の明確なエラーメッセージ
- 不正な指定形式: "invalid quantization" 相当の明確なエラーメッセージ
